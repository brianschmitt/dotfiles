## Installation

On Windows:
```shell
git clone https://github.com/brianschmitt/dotfiles

cd dotfiles

setup.bat
```

On Linux:
```shell
git clone https://github.com/brianschmitt/dotfiles

chmod +x dotfiles/setup.sh

./dotfiles/setup.sh
```
