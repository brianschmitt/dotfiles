## Installation

On Windows:
```shell
git clone https://github.com/brianschmitt/dotfiles

cd dotfiles

setup.bat
```

On Mac:
```shell
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
brew install git
git clone https://github.com/brianschmitt/dotfiles
chmod +x dotfiles/setup.sh
./dotfiles/setup.sh
./dotfiles/brew.sh
./dotfiles/osx.sh
```

On Linux:
```shell
git clone https://github.com/brianschmitt/dotfiles

chmod +x dotfiles/setup.sh

./dotfiles/setup.sh
```
